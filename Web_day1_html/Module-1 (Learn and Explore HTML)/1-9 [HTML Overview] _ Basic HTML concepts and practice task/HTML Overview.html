HTML Overview

HTML (Hypertext Markup Language) is the standard markup language for creating web pages. It is used to structure content on the web and consists of a series of elements that define the structure and layout of a webpage.

HTML elements are represented by tags, which are enclosed in angle brackets. For example, the <p> tag is used to create a paragraph, while the <a> tag is used to create a hyperlink.

HTML documents are structured in a hierarchical manner, with a root element (<html>) that contains a head section (<head>) and a body section (<body>). The head section contains metadata about the document, while the body section contains the visible content of the webpage.

Basic HTML Structure

The basic structure of an HTML document includes the following elements:

  • <!DOCTYPE html>: Declares the document type and version of HTML being used.
  • <html>: The root element that contains all other HTML elements.
  • <head>: Contains metadata about the document, such as the title, character set, and links to stylesheets.
  • <body>: Contains the content of the web page, including text, images, links, and other elements.
