HTML List

Lists are used to group related items together. There are three types of lists in HTML: ordered lists, unordered lists, and definition lists.

    • Unordered List

    Unordered lists are used to create a list of items that do not have a specific order. The items in an unordered list are typically displayed with bullet points.

    Unordered lists are created using the <ul> tag, and each item in the list is created using the <li> tag.

    For example:

    • Birayni
    • Chicken
    • Fish
    • Vegetable
    • Fruit
    • Ice Cream
    • Chocolate
    • Cheese
    • Butter

    Line Break

    Line breaks are used to create a new line in a block of text. Line breaks are created using the <br> tag.

    For example:

    This is the first line.
    This is the second line.

    This is the third line.
