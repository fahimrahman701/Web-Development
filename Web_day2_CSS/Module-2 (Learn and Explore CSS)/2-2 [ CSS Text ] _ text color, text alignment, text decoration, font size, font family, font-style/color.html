CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It allows you to control the layout, colors, fonts, and overall appearance of web pages.

CSS consists of a set of rules that define how elements should be displayed on the screen. Each rule consists of a selector (which specifies the element to be styled) and a declaration block (which contains one or more declarations that define the styles to be applied).

Basic CSS Rule

The basic structure of a CSS rule includes the following components:

  • Selector: Specifies the HTML element(s) to be styled.
  • Declaration Block: Contains one or more declarations enclosed in curly braces.
  • Declaration: Consists of a property and a value, separated by a colon.
