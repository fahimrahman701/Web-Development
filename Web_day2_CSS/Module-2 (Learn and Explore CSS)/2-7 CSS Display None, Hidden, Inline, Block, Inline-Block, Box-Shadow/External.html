CSS Display

The display property specifies the display behavior of an element.

  • block: The element is displayed as a block element (like a <div>).
  • inline: The element is displayed as an inline element (like a <span>).
  • none: The element is not displayed at all.

CSS Box Shadow

The box-shadow property adds shadow effects around an element's frame.

  • box-shadow: 10px 10px 5px #888888;
  • This creates a shadow that is offset by 10 pixels to the right and down, with a blur radius of 5 pixels.

CSS Positioning

The position property specifies the type of positioning method used for an element.

  • static: Default value. Elements are positioned according to the normal flow of the document.
  • relative: Element is positioned relative to its normal position.
  • absolute: Element is positioned relative to its nearest positioned ancestor.
  • fixed: Element is positioned relative to the viewport, which means it always stays in the same place even when scrolled.
